# Tienda de Pasteles - Proyecto Web

Mini-aplicación web interactiva para la gestión de pedidos de una pastelería.

## Requerimientos cumplidos
- **Estructura:** HTML, CSS y JS separados.
- **Interacción:** Manipulación del DOM, eventos de formulario y creación dinámica de elementos.
- **Validaciones:** 1. Campo nombre (mínimo 3 caracteres).
    2. Campo cantidad (debe ser mayor a 0).
    3. Validación de correo electrónico (presencia de '@').
    4. Prevención de envío con `preventDefault`.
    5. Mensajes de feedback visual en pantalla.
- **Persistencia:** Uso de `localStorage` para guardar pedidos.
- **Calidad:** Código ordenado y diseño responsivo.

## Instrucciones de ejecución
1. Clonar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador moderno.
3. Ingresar los datos en el formulario y hacer clic en "Hacer Pedido".
